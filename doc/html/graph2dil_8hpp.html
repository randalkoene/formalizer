Detailed Description

This header file is used for declarations specific to the graph2dil tool.

Functions and classes available here are typically useful to create the data structures needed when producing DIL Hierarchy compatible output from Graph data.

Versioning is based on https://semver.org/ and the C++ header defines __GRAPH2DIL_HPP.
