std::time_t fz::ReferenceTime::Time ( )

Report the current time according to the ReferenceTime frame being used.

Note: This function returns actual time event for emulated_time == 0. It is a precaution, in case unspecified time values are sometimes left zero-valued. (This is subject to change and may be set to emulated_time > RTt_unspecified.)

Returns
current time, either emulated as specified or actual.
