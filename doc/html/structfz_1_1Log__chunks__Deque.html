Detailed Description

Log chunks (deque list)

A deque list of smart pointers to Log chunks. This is how chunks are connected in the Log data structure.

Get index of Log chunk pointer by ID key. Returns size() if not found.

Find index of Log chunk from its ID.

This implementation attempts to be quick about it by relying on the sorted order of Log chunks to apply a quick search method. (This is actually probably the same as using std::binary_search.)
