Detailed Description

Standardized Formalizer Log chunk ID key.

The principal constructors (all but the default constructor) each call a validity test for the key format and can throw an ID_exception.

In various containers, the ordering of Log chunk ID keys is determined by the provided operator<(). It calls its equivalent in Log_TimeStamp, where std::tie() enables lexicographical comparison, from the largest temporal component to the smallest.

The default constructor is provided for various special use cases such as initialization of containers. The isnullkey() function can test for this special state.
