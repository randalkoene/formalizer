void VTD_Placer::place ( )
virtual

Place VTD Nodes.

The resulting map may require that the target dates of some VTD Nodes that cannot be achieved are modified.

EPS_map.group_and_place_movable() is called with include_UTD=false so that only VTD Nodes and Nodes inheriting target dates are placed.
