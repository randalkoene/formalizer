◆ map_exact_target_date_entries()

bool schedule::map_exact_target_date_entries ( )

Note: When an exact target date Node has been partially completed, i.e.

minutes_to_complete() is less than get_required_minutes(), then, if the current time is later than the start time of the exact time interval, then it makes sense that the remaining time belongs at the end of the interval. Otherwise, we can map it to the start of the interval (e.g. to complete a meeting early).
