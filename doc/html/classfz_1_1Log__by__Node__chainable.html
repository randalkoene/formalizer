void fz::Log_by_Node_chainable::set_Node_prev_ptr ( Log_chunk *  prevptr)

Set the previous in chain-by-Node based on a Log chunk pointer.

This assumes that you know that the pointer is correct and that it is indeed a chunk and not an entry. No confirmations are performed here. The target is set accordingly, both ID key and rapid-access pointer.

A nullptr can be given to clear (by)node_prev.

Parameters
prevptra valid pointer to previous chunk in the chain (or nullptr).
