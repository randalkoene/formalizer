Detailed Description

This structure is used to collaboratively maintain a stack trace for use in both logged errors and exception handling.

Functions that wish to expose their place on the stack for easy tracing should use the defined ERRSTACK_ macros to collaboratively manage this stack.

For more backtground information, read the card at https://trello.com/c/lNwxlrbT.
