Detailed Description

This is the data structure used for elements of the request stack for Graph modification.

Each part of the modification requested is represented by one of these elements, including the data that is pointed to by one or more of the shared memory pointers.

For more information about Edit_flags and Edit protocols, see: https://trello.com/c/ooCyccJ0/95-fzedit-node-and-edge-editor#comment-5fc059eee4536d8a147db56f
