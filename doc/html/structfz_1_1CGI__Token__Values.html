Detailed Description

Find GET or POST token-value pairs and make them available as maps.

See for example how this is used in fzlogtime.

Note that has_query_string and has_post_length report environment variables found. The actual number of token-value pairs found is reported by GET.size() and POST.size(). After merge() is called POST.size() reports the merged number of unique token-value pairs.
