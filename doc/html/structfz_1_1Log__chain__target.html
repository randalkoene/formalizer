Detailed Description

This data structure is used to follow a chain by-Node through the Log that can lead to Log chunks or Log entries.

The two possible IDs and two possible pointers are in unions to share the same memory space.

Parameters
ischunkflag is true if the target is a Log chunk, false if Log entry.
chunkidentifies and points to a Log chunk (or is nullkey & nullptr).
entryidentifies and points to a Log entry (or is nullkey & nullptr).
