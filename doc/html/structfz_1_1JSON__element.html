◆ find_one_of()

bool fz::JSON_element::find_one_of ( JSON_element_data_vec &  buffer_list)
inline

Compare the label of this JSON element with labels in buffer_list.

As soon as one matches, copy the value of this element to the JSON_element_data of that item in buffer_list, thereby returning the value for the label that the item was looking for.
