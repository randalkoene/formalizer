Set-up and Replicate System Envionment

To set up and replicate a complete system environment, including the Formalizer, on a new machine or VM:

  1. Make sure you recently backed up the Formalizer database.

  2. Make sure you recently created or synchronized a backup partition.

  3. Make sure your backup drive also contains a copy of the replicate.py script.

  4. Make sure your backup drive contains a copy of the .fzdatabase-token file.

Then:

  1. Install the operating system on the machine/VM with a user account with identical username.

  2. Attach the backup drive and copy replicate.py and .fzdatabase-token to the home directory.

  3. Run ./replicate.py <path-to-mountable-partition> <path-to-mountpoint>

If replicate.py exits with an error then try to correct the error and then simply run the same replicate command again.
