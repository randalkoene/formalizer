Tips

  1. Download the presentation as PDF.
  2. Move the PDF to a new folder to keep related things in one place.
  3. In that folder, use `thumbpdf` to create the thumbnail images. See example below and notes.

Example:

		thumbpdf --nomakepdf onboarding-slides.pdf

Notes:

  • Without `--nomakepdf` the thumbnails are placed into a new PDF and the image files are deleted.
  • You may need to rename the PDF, because `thumbpdf` is pretty finicky about file names.
  • The thumbnails may be too small at the default resolution (9 == 90 pixels wide). For example, try `--resolution 32` (320px wide) or `--resolution 64` (640px wide).
