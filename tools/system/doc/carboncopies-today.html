Demos

To build an example, from Netmorph reservoir generation, through Connectome tuning to hidden function, to virtual data collection and Neuroglancer inspection:
  • Access the VM, run API and NES.
  • Pick the example (e.g. xor_scnm) and request a full run-through: ./Run.sh -R -n -S -M
  • Inspect in Neuroglancer: Dataset in Neuroglancer (API server must be running.)
  • Or, for the autoassociative memory example, in Neuroglancer: Dataset in Neuroglancer
For Blender demo of Netmorph generated example:
  • Open Blender from its folde in Downloads.
  • Open the example .blend.gz for the xor_scnm example from the CCF-backup/braingenix folder on the SSD.
  • Use the View/Navigate menu to pick Fly Through
  • To create a Blender file of the reservoir for the Autoassociative memory example: ./autoassociative_reservoir.py -modelfile nesvbp-autoassociative -DoBlend -BevelDepth 0.2 (BlenderExec was already pointing at the right executable)
