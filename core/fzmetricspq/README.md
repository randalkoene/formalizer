# fzmetricspq - Access to the Metrics tables in the database

This program provides access to read and write the metrics
tables stored in the Formalizer database.

Generic data acess is also provided by fzmetricspq. See,
for example, how this is used by dbjsondecrypt.js.

When writing to a table that does not exist yet, the table is
automatically created.
